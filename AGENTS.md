# Project Goal

Build a credit card fraud detection system using:

- Python Flask backend
- React frontend with CSS
- Kartik Shenoy fraud detection dataset
- XGBoost model

# Input Features

- amount
- category
- gender
- age
- city population
- state
- job
- city
- hour
- distance

# Rules

- Keep the UI simple and clean
- Use functional React components
- Use CSS only
- Backend should expose `POST /predict`
- Save trained model as `fraud_model.pkl`
- Save preprocessing artifacts as `encoders.pkl`
